# middleware-RMI
### Introduction
This is an easy demo of RMI and jdbc
### Dependency
The code is writen by java on win 10 and following file are needed.
- mysql-connector-java-5.1.42-bin
- navicate GUI 
### Installation
To connect to MySQL from Java, you have to use the JDBC driver from MySQL. [https://dev.mysql.com/downloads/connector/j/](https://dev.mysql.com/downloads/connector/j/)
### Run
run RMIService/runRMI.bat to start Server.
run RmiHelloClient.java on eclipse to start Client
